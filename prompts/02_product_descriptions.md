## 2.1 B2B产品描述（工厂→采购商）

```
你是一位专业的B2B产品文案专家，精通制造业采购心理。

产品信息：
- 产品名称：[INSERT_PRODUCT]
- 核心规格：[INSERT_SPECS]
- 认证：[INSERT_CERTS]（如：ISO9001/CE/FCC/UL）
- 年产能：[INSERT_CAPACITY]
- 最小起订量：[INSERT_MOQ]
- 付款方式：[INSERT_PAYMENT]
- 交期：[INSERT_LEADTIME]
- 目标市场：[INSERT_MARKET]

目标读者：[INSERT_BUYER_ROLE]（如：采购经理/产品经理/CEO）
他们最关心：[INSERT_PRIMARY_CONCERN]（如：价格/质量/交期/产能）

写作框架（B2B采购心理）：

**标题**：[核心关键词] - [差异化卖点] | [工厂名称]
- 公式：产品名 + 数字量化优势 + 品牌背书

**第一段 - 钩子**（3句话）：
"贵司正在寻找[客户痛点]的[产品类别]，我们专注这个领域[X]年，服务过[数字]家[类似客户]，包括[知名客户类型]。"

**第二段 - 核心价值**（4-5句话）：
- 规格参数（用表格或列表）
- 认证优势（为什么重要）
- 产能优势（能承接多少量）
- 质量控制（具体流程）

**第三段 - 信任背书**（2-3句话）：
- 客户案例（数字量化）
- 合作过的品牌（知名但不泄密）

**第四段 - CTA**：
"想了解完整规格书和样品报价？[链接/联系方式]"

请按以上框架撰写完整B2B产品描述，300-500词，英文。
```

## 2.2 B2C电商平台描述（Amazon）

```
平台：[Amazon / Etsy / Shopify / TikTok Shop]
产品类型：[INSERT_PRODUCT_TYPE]

产品信息：
- 产品名：[INSERT_NAME]
- 主要卖点：[INSERT_KEY_FEATURE]
- 材质/成分：[INSERT_MATERIAL]
- 尺寸/规格：[INSERT_DIMENSIONS]
- 颜色：[INSERT_COLORS]
- 目标用户：[INSERT_TARGET_AUDIENCE]
- 使用场景：[INSERT_USE_CASE]
- 竞品价格区间：[INSERT_PRICE_RANGE]
- 我的价格：[INSERT_MY_PRICE]
- 我的优势：[INSERT_EDGE]

写作策略：

**5点描述（Bullet Points）**：
每个bullet格式："[关键词] | [具体说明] | [给买家什么]"

Point 1 - 材质/质量：
"[PREMIUM_MATERIAL], engineered for [SPECIFIC_BENEFIT]. Unlike common alternatives, [OUR_DIFFERENCE]."

Point 2 - 功能/用途：
"[KEY_FUNCTION] made simple. Whether you're [USE_CASE_1] or [USE_CASE_2], this delivers [RESULT]."

Point 3 - 差异化：
"Here's what sets us apart: [1], [2], [3]. Most competitors only offer [THEIR_LIMITATION]."

Point 4 - 买家安心：
"[X]-day warranty | [Y]reason returns | [Z]h customer support. Your purchase is 100% protected."

Point 5 - 场景/生活方式：
"Perfect for [SCENE_1], [SCENE_2], or [SCENE_3]. Whether you're [PERSONA_A] or [PERSONA_B], this is made for you."

**产品描述正文**（150-200词）：

开头用"谁应该买这个"引入：
"Looking for [PROBLEM_SOLUTION]? This is exactly what [TARGET_AUDIENCE] have been searching for."

中间用2-3个场景描述+具体参数
结尾用QA格式解答常见顾虑

请按以上框架生成完整的Amazon listing内容（标题+5点+描述）。
```

## 2.3 多语言产品描述

```
你是一位多语言电商文案专家。

请将以下产品描述翻译并本地化到：

1. 西班牙语（拉美市场）
2. 葡萄牙语（巴西市场）  
3. 法语（非洲法语区）
4. 德语（欧洲市场）
5. 阿拉伯语（中东市场）

原产品描述：
[INSERT_DESCRIPTION]

本地化要求：
- 不要直译，要符合当地表达习惯
- 调整文化敏感内容
- 保留关键销售信息
- 长度控制在原文80-120%范围内
- 标注需要注意的文化/法律差异

额外要求：
- 阿拉伯语版本要从右到左排版
- 德语版本要使用正式称呼
- 西语版本要区分拉美和西班牙当地用语差异

请提供每个语言版本的描述。
```

## 2.4 差异化卖点提炼

```
产品：[INSERT_PRODUCT]
当前市面常见卖点：[INSERT_COMPETITOR_CLAIMS]
我的产品优势：[INSERT_MY_ADVANTAGES]

请用以下框架帮我们提炼3个核心差异化卖点：

**卖点提炼框架**：

卖点1：[SCADA Framework]
- S（Specific）：具体而非笼统
- C（Concrete）：可以验证的
- A（Audience）：对目标客户重要
- D（Different）：竞品没有/做不到
- A（Authentic）：我们真实能做到

卖点2：[Before-After Bridge]
- Before（客户现在的痛）：
"Most [PRODUCT_CATEGORY] buyers struggle with [PROBLEM]..."
- After（使用后的改变）：
"...and end up [NEGATIVE_OUTCOME]. But those who find the right solution achieve [POSITIVE_OUTCOME] in just [TIMEFRAME]."
- Bridge（我们怎么做到的）：
"[OUR_PRODUCT] is built specifically to solve [SPECIFIC_PROBLEM] because [OUR_UNIQUE_APPROACH]. The result: [QUANTIFIABLE_IMPROVEMENT]."

卖点3：[Ego-Aligning]
- 目标客户的身份认同：[INSERT_IDENTITY]
- 他们想被看作：[INSERT_HOW_THEY_WANT_TO_BE_SEEN]
- 我们的产品如何帮他们彰显这个身份：
"[PRODUCT] is for [IDENTITY] who refuse to settle for [COMPETITORS'_WEAKNESS]. It's for those who [ASPIRATIONAL_STATEMENT]. Because you deserve [VALUEOUTCOME]."

请输出3个卖点，每个卖点包含：
- 一句话核心主张
- 支撑的具体证据/数据
- 适合出现在描述的哪个位置（标题/副标题/正文/CTA）
```

## 使用说明

1. 根据目标平台（B2B/B2C）选择对应模板
2. 填入实际产品信息
3. B2B强调数据/认证/产能，B2C强调场景/感受/生活方式
4. 写完后用humanizer检查是否有AI痕迹
5. 定期A/B测试不同版本
