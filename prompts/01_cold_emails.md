## 1.1 首次接触 - 通用版

```
你是一位有15年经验的外贸销售专家，擅长写高回复率的开发信。

请为以下场景撰写一封开发信：

产品/服务：[INSERT_PRODUCT]
目标客户职位：[INSERT_TITLE]（如：采购经理/CEO/运营总监）
客户行业：[INSERT_INDUSTRY]
公司规模：[INSERT_SIZE]
我的公司定位：[INSERT_POSITIONING]

要求：
1. 标题抓眼球，前20字决定打开率
2. 正文4段以内，总词数80-120
3. 开头用具体数据或场景钩住客户（如"我注意到贵司最近在拓展[市场]..."）
4. 中间说明我能提供什么具体价值（不是泛泛的"质量好价格优"）
5. 结尾有明确CTA（不是"期待您的回复"这种废话）
6. 语气：专业但有温度，像真人写的
7. 写完后标注哪些地方需要客户自己替换[方括号内容]

请分别写3个版本：
A. 简洁直接型（适合德/美/英客户）
B. 关系导向型（适合印/巴/东南亚）
C. 数据驱动型（适合大公司采购）
```

## 1.2 首次接触 - 服装/纺织行业

```
角色：资深外贸服装订单专家，擅长开发欧美时尚品牌客户

产品信息：
- 主营：[休闲装/正装/运动装/童装]
- 面料：[有机棉/再生涤纶/麻]
- 认证：[GOTS/OCS/BSCI/SEDEX]
- MOQ：[INSERT_MOQ]
- 起订量单件价：[INSERT_PRICE]
- 交期：[INSERT_LEADTIME]

目标客户：
- 品牌名：[INSERT_BRAND]
- 采购负责人：[INSERT_NAME]
- 采购量级：[INSERT_VOLUME]

撰写要点：
1. 开头提及该品牌最近的市场动作或新品（显示做过功课）
2. 中间用"三明治法则"：夸对方+自我介绍+给个钩子
3. 具体说明我们的差异化（不是"我们质量好"而是"我们专做[品类]12年，服务过XX个同类品牌"）
4. 附上一个无法拒绝的offer（如：新客户首单10%折扣/免费色彩咨询/免费开发样）
5. CTA明确：约15分钟电话而不是"期待回复"

写2个版本：邮件版（150词内）+ LinkedIn连接消息版（300字符内）
```

## 1.3 首次接触 - 电子/3C行业

```
角色：消费电子外贸专家，精通跨境电商供应链

产品线：[智能家居/音频设备/充电配件/穿戴设备]
目标市场：[INSERT_MARKET]
我的优势：[INSERT_ADVANTAGE]（如：工厂直接价/专利齐全/可小批量定制）

目标客户背景：
- 销售平台：[Amazon/eBay/独立站/线下商超]
- 年GMV：[INSERT_GMV]
- 寻找供应商原因：[开拓新品/价格压力/现有供应商出问题]

撰写策略：
1. 用"痛点+解决方案"框架
2. 痛点要具体（如："我们很多客户反映C9芯片缺货导致listing下架..."）
3. 解决方案要可验证（如："我们在深圳有现货，最快3天到美国仓"）
4. 报价要聪明（不直接报价，而是"根据我们的经验，您这类产品起订量单件可以做到$XX"）
5. 制造稀缺感（如："我们这个月只能接3个新客户"）

写1封邮件 + 1条Amazon卖家群发模板
```

## 1.4 首次接触 - 机械/工业设备

```
你是一位重工业设备外贸经理，曾帮助多家中国工厂进入ABB/西门子/施耐德的供应链。

产品：[INSERT_PRODUCT]（如：CNC机床/注塑机/激光切割机/包装设备）
目标客户：[INSERT_COMPANY]（行业龙头或细分市场前三）
采购背景：设备更新周期/产能扩张/中国供应商替换

写作要求：
1. 研究客户公司背景，找出具体采购需求
2. 展示我们解决过同类问题的案例（用数字量化）
3. 证书/资质要放在最能建立信任的位置
4. 报价策略：报范围价（如"$50,000-$80,000根据配置定"）
5. 结尾提供免费的"采购需求分析"服务作为钩子

写1封高意向开发信 + 1份附件报价单模板说明文字
```

## 1.5 展会后跟进

```
场景：刚参加完[广交会/Canton Fair/德国科隆/美国CES]展会

展会收获：
- 交谈过的客户：[INSERT_CLIENTS]
- 交换的名片数：[INSERT_COUNT]
- 主要需求：[INSERT_NEEDS]
- 价格范围：[INSERT_PRICE_RANGE]

跟进策略（按客户分类）：
A类（当场表现出浓厚兴趣/要了样品）：3天内发邮件
B类（正常交谈/有兴趣但没承诺任何事）：7天内发邮件  
C类（只换了名片/泛泛交流）：14天内发邮件

对于A类客户：
- 邮件结构：展会上最难忘的瞬间 + 具体讨论过的内容 + 下一步行动（样品/视频/报价）
- 语气：像老朋友重逢，而不是销售追单

对于B类客户：
- 邮件结构：展会回顾 + 你们能给什么 + 一个无法拒绝的理由（限时优惠/名额有限）
- 重点：不要用"期待您的回复"——用具体问题引导回复

对于C类客户：
- 邮件结构：简短提醒 + 价值补充（如"展会期间没来得及介绍我们的[新品]..."）
- 目标：激活对话，不要急于成交

请为以上3类客户各写1封跟进邮件。
```

## 使用说明

1. 复制对应Prompt到ChatGPT/Claude
2. 替换所有[INSERT_XXX]为你的实际信息
3. 根据客户类型选择合适版本（A/B/C）
4. 发送前检查：是否像真人写的？是否有具体数字/名称？
5. 避免：直接发送，多个版本中选择最适合的
